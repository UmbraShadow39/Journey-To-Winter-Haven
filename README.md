# Journey To Winter Haven

Journey To Winter Haven is a text-based RPG where you battle monsters, gain levels, unlock titles and find rare loot. The game culminates in one major decision that will affect your child's destiny. What path will you choose -- good or evil?

## Features

- Turn-based combat with attack, defense, AP, and special moves  
- Unique monsters, each with special behaviors and attack patterns  
- Level-up system with stat increases and full heals  
- Potions and consumables to manage survival  
- Titles and achievements based on your actions  
- Monster essence collection influencing story progression  
- Multiple hidden endings based on your decisions, performance, and morality  
- Story-driven narrative with branching choices and replay value

## How to Play

### Windows Executable (Recommended)

1. Download the `.exe` from the Releases page.  
2. Double-click to run.  
No Python installation required.

### Running From Source (Advanced Users)

If running the Python file directly:

```
python Journey_To_Winter_Haven_.1.py
```

Built using **Python 3.11**.
